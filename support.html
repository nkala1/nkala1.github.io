Drive School Manager Privacy-Policy

Drive School Manager is an app for managing driving lesson scheduling between driving instructors and students.

The app may collect the following information:

- Name - Email address - Phone number - User role, such as instructor or student - Lesson scheduling information - Lesson status information - Invite code usage information

  • Name
  • Email address
  • User role<, such as instructor or student/li>
  • Lesson scheduling information
  • Lesson status information
  • Invite code usage information

How We Use Information

  • Create and manage user accounts
  • Connect students with their driving instructor
  • Manage instructor availability
  • Allow students to request lessons
  • Allow instructors to approve or reject lesson requests
  • Display lesson status and lesson history
  • Support basic app functionality

Data Storage

  • Drive School Manager uses Firebase Authentication and Firebase Firestore to store account and scheduling information.

Data Sharing

  • We do not sell user data. We do not use user data for advertising or tracking.

Data Access and Deletion

  • Users may request account or data deletion by contacting the app support email.
